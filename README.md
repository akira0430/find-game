1.ゲームを紹介するサイトです

タイトル、ゲームの個数を投稿

メールの送信、電話番号、詳細情報、ゲーム名、ゲームの画像を投稿できます

投稿した本人のみが投稿の削除をおこなったり編集できるようにしました。閲覧はどのユーザーでもできます


・ajax送信
・サムネイルのの画像の表示の仕方
・scaffoldでのcontrollerの記載の仕方
